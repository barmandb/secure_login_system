# 🔐 Secure Login System (Flask)

A secure authentication system built using Flask with modern UI and OTP verification.

## 🚀 Features
- User Registration & Login
- Password Hashing (bcrypt)
- Email OTP Verification
- Secure Dashboard
- Password Reset System
- Modern UI (Glassmorphism style)

## 🛠️ Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS
- Flask-Login
- Flask-Bcrypt
- Flask-Mail
